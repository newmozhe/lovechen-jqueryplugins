普通模式：

$("#txt_date").lidate();

指定日历中星期的第一天：

$("#txt_startofweek").lidate({ weekDisplay:["二","三","四","五","六","日","一"], weekCss:[null,null,null,null,"lidate_holiday","lidate_holiday",null], startOfWeek:2 });

仅允许选择指定的日期：

$("#txt_mode1").lidate({ mode:1, flagDays:[1,3,5,7,9] });

仅允许选择未指定的日期：

$("#txt_mode2").lidate({ mode:2, flagDays:[1,3,5,7,9] });

English：

$("#txt_english").lidate({ monthDisplay:["January","February","March","April","May","June","July","August","September","October","November","December"], weekDisplay:["Sun","Mon","Tue","Wed","Thu","Fri","Sat"], });

日历生成及日期点击事件：

$("#txt_event").lidate({ afterDraw:function(obj){ //演示动态设置日期标签 obj.opts.flagDays = [20,21,22,23,24,25,26]; obj.setFlagDay(); }, onChoose:function(obj,date,fmt){ //日期选择后执行自定义事件，并阻止默认事件 $(obj.ele).val("星期" + date.getDay()); return false; } });
